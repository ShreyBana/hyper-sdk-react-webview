# hyper-sdk-react-webview

Custom webview for a hyper-sdk webview integration.

## Installation

```sh
npm install hyper-sdk-react-webview
```

## Usage

```js
import { multiply } from 'hyper-sdk-react-webview';

// ...

const result = await multiply(3, 7);
```
